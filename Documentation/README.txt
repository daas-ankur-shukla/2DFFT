README FILE

Application of 2-D Fourier Transform & Butterworth/Gaussian Low Pass Filters on an Image.


GENERAL USAGE NOTES            25-October-2016
----------------------

Minimum System Requirements :

 - Operating System : Windows, Linux, Mac OS
 - Compatible with MATLAB 2015b & Above.
 - Application is backward incompatible.


Steps :

1. Browse to load your image.
2. Press on the 'Compute 2-D FFT' Button to compute the Fourier Transform of the input image.
3. Select a filter type.
4. Provide necessary inputs based on Filter selected in step 3.
5. Press on 'Apply Filter'.
6. To download the output images to the source folders of the images, click on the 'Download Response' button.
